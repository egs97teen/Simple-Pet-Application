About This Application

This is a simple pet application built with HTML, CSS, and JavaScript. Created with minimal time, this application allows users to create a visual list of pets. All functionality of this site is built using JavaScript and jQuery.

If given more time, this site would prove to be more efficient if given a proper back-end and non-relational database. I would also like to refactor this project using React.js as a front-end framework for a more fluid and optimized user-experience. I also wouldn't use annoying alert messages!

Created by Eric Sagata
